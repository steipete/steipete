# Vision

This repository is the public GitHub profile for Peter Steinberger. The README should stay a concise, current map of who Peter is, what he is building, and where people can find the canonical projects.

## Priorities

- Keep identity, role, project names, ownership, status, and links accurate.
- Prefer canonical project domains or current repository URLs over redirects and former names.
- Preserve Peter's direct voice and editorial control; additions should improve the profile, not merely make it busier.
- Use coarse, verifiable thresholds for volatile numbers such as stars so they remain truthful between refreshes.

## External Content

External images and live statistics must earn their reliability and privacy cost. They should use public data, label metrics accurately, render correctly through GitHub and its image proxy, and come from a service with a credible maintenance story. Do not combine public and private-derived values under a misleading label.

These boundaries capture the contribution-graph decisions in [PR 15](https://github.com/steipete/steipete/pull/15) and [PR 16](https://github.com/steipete/steipete/pull/16).

## Non-goals

- An automated or exhaustive inventory of every repository.
- Vanity metrics that sacrifice accuracy, privacy, or clarity.
- A release or package lifecycle; this repository is profile content.
